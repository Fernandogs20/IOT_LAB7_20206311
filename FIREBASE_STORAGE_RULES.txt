REGLAS DE FIREBASE STORAGE (Cópialas exactamente en Firebase Console)
========================================================================

1. Abre Firebase Console: https://console.firebase.google.com
2. Selecciona tu proyecto: "lab520206311"
3. Ve a: Storage > Reglas
4. Reemplaza TODO el contenido con esto:

---BEGIN RULES---
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    // Permitir lectura pública de cualquier archivo
    match /{allPaths=**} {
      allow read: if true;
    }

    // Permitir a usuarios autenticados subir archivos en sus propias carpetas
    match /profiles/{userId}/{fileName} {
      allow write: if request.auth.uid == userId;
      allow read: if true;
    }

    match /tareas/{userId}/{document=**} {
      allow read, write: if request.auth.uid == userId;
    }

    // Cualquier otro archivo: solo lectura pública
    match /{document=**} {
      allow read: if true;
      allow write: if false;
    }
  }
}
---END RULES---

5. Haz clic en "Publicar"
6. Verifica que diga "Las reglas se publicaron correctamente"

¿Lo hiciste? Responde SÍ o NO y luego te doy los comandos para reconstruir la app.
